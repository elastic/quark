NAME

quark-btf — query quark BTF offsets

SYNOPSIS

quark-btf [-bv] [targets ...]

quark-btf [-bv] [-f btf_file]

quark-btf [-v] [-f btf_file] [-g btf_name]

DESCRIPTION

The quark-btf program prints out the kernel structures offsets used by quark.

If operands are given, the output is curbed for those specified targets. The exit status is the number of failed targets.

The options are as follows:

-b
Also print the offset in bits.
-v
Increase quark_verbose, can be issued multiple times.
-f btf_file
Print all offsets quark would use from btf_file.
-g btf_name
Generate the internal btf C structure used by quark. Must be used together with -f btf_file.

This is only used to generate btfhub.c via genbtf.sh, and since btfhub-archive never changes, chances are you'll never need this. btf_file.

EXIT STATUS

quark-btf exits with number of targets failed if targets is specified. Otherwise 0 in case of success.
