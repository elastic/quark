QUARK_QUEUE_CLOSE(3) Library Functions Manual QUARK_QUEUE_CLOSE(3)

NAME

quark_queue_close — closes a quark_queue

SYNOPSIS

#include <quark.h>

int
quark_queue_close(struct quark_queue *qq);

DESCRIPTION

quark_queue_close closes the quark_queue pointed to by qq, which must have been previously opened with quark_queue_open(3).

RETURN VALUES

Zero on success, -1 otherwise and errno is set.
