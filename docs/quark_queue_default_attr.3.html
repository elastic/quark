NAME

quark_queue_default_attr — fetch default attributes for quark_queue_open(3)

SYNOPSIS

#include <quark.h>

void
quark_queue_default_attr(struct quark_queue_attr *attr);

DESCRIPTION

quark_queue_default_attr copies out default attributes for opening a quark_queue.

Refer to quark_queue_open(3) for an in depth explanation of each member.
